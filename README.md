# Windows-EventLog-Analysis

## Objective
To analyze Windows Security logs using Event Viewer and identify successful and failed login attempts.

## Tools Used
- Windows Event Viewer
- Windows Security Logs

## Event IDs Analyzed
- 4624 – Successful Login
- 4625 – Failed Login

## Activities Performed
- Opened Event Viewer
- Navigated to Windows Security Logs
- Investigated login-related events
- Identified successful and failed authentication attempts
